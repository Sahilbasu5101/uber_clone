cors package :- configure Cross-Origin Resource Sharing (CORS) in Express or Connect applications.By default, web browsers block frontend JavaScript (like fetch or XMLHttpRequest) from reading responses from a different origin (domain, protocol, or port) due to the Same-Origin Policy. The cors package provides a simple way to bypass this restriction by setting the appropriate HTTP response headers, such as Access-Control-Allow-Origin
